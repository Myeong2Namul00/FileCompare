﻿# (C# 코딩) 파일 비교 프로그램


## 개요
- C# 프로그래밍 학습
- 1줄 소개: 두 폴더의 파일/폴더를 비교하고 색상으로 차이를 확인하며 양방향 복사가 가능한 프로그램
- 사용한 플랫폼:
    - C#, .NET 10 Windows Forms, Visual Studio, GitHub
- 사용한 컨트롤: 
    - SplitContainer, ListView, TextBox, Button, Panel, Label, FolderBrowserDialog
- 사용한 기술과 구현한 기능:
    - 두 개의 폴더 경로를 선택하고 각각 목록 로드
    - 동일 이름 항목을 좌/우 한 줄에 정렬하여 비교
    - 파일/폴더 상태에 따른 색상 표시(동일/최신/구버전/한쪽만 존재)
    - 폴더 항목 Bold 표시 및 폴더 통합 비교
    - 선택 항목 단위 복사(파일/폴더 모두 지원)
    - 폴더 복사 시 하위 구조 전체 재귀 복사
    - 오래된 항목으로 덮어쓸 때 경고 메시지 표시
